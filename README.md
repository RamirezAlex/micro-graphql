# micro-graphql
> Repo meant to be cloned for our GraphQL microservices

### Usage
 + clone the repo and name it
```bash
git clone https://github.com/hyperfuse/micro-graphql.git [name]
```

+ install dependencies
```bash
 yarn
```

+ start microservice
```bash
 yarn start
```


+ Make it your own!
